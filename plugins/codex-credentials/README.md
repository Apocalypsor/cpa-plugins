# Codex Credentials

CPA plugin page for inspecting Codex OAuth credentials.

Open:

```text
/v0/resource/plugins/codex-credentials/index.html
```

It shows each Codex credential's email, team, plan, account id, status, and
whether it needs login. Login buttons use CPA's built-in Codex OAuth flow and
save a real credential for the logged-in account.
